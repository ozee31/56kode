# Claude Design: Use Cases for Marketing & Localization

This article explores how Claude Design can be used by marketing and localization teams to streamline visual content creation. Claude Design is a tool that allows users to create visual content through a chat interface, leveraging a design system built from existing codebases and design files.

## Key Features

*   **Design System Integration:** Claude Design automatically reads codebases and design files to build a design system, ensuring consistent branding.
*   **Collaboration:** The tool allows for internal sharing and collaborative editing.
*   **Inline Comments:** Users can leave comments directly on elements within the canvas for precise feedback.

## Use Cases

1.  **Visual Style Guides:** Create easy-to-digest visual style guides for local markets, including tone examples and key terms.
    ```
    Prompt: "Create a one-pager style guide for translating a mobile app UI into Polish. Include tone examples, key terms, and what to avoid."
    ```

2.  **Training Decks:** Generate training slides quickly by providing the tool with training content. Claude structures the deck, saving time on manual slide creation.

3.  **Localized Marketing Visuals:** Adapt marketing visuals for different markets by uploading an existing image and asking Claude to modify it for a specific audience.

## Limitations

*   Inline comments may occasionally disappear.
*   Large codebases can cause lag.

## Conclusion

Claude Design is a powerful tool for creating and adapting visual content, offering time-saving features and collaborative capabilities for marketing and localization teams.
